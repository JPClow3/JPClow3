# Code of Conduct

## Nosso compromisso

Este projeto busca um ambiente aberto e respeitoso para todos.

## Comportamento esperado

- Use linguagem respeitosa e inclusiva.
- Aceite feedback construtivo.
- Foque em melhorar o projeto e a colaboração.

## Comportamento inaceitável

- Assédio, ofensas pessoais ou discriminação.
- Divulgação de dados privados sem consentimento.
- Condutas que prejudiquem a comunidade.

## Aplicação

Casos de violação podem ser reportados por meio do contato no README.
